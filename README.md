## Euler's Totient Function
### Concept
Number of co-prime pairs from 1 to N (1,N; 2,N...) for a number N
Φ(1) = 1
Φ(2) = 1
Φ(10) = 4
### Approach
### Complexities
#### Time
O(n(log(n)))
#### Space
O(n)
